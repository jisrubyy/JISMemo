# JISMemo Development Log / JISMemo 개발일지

## Version 1.3 (2024-12-19) / 버전 1.3 (2024-12-19)
### Bug Fixes & Improvements / 버그 수정 & 개선
- Fixed text paste functionality from other programs / 다른 프로그램에서 복사한 텍스트 붙여넣기 기능 수정
- Resolved text input issues after image pasting / 이미지 붙여넣기 후 텍스트 입력이 안되는 문제 해결
- Added multiple image support and individual image deletion / 여러 이미지 지원 및 개별 이미지 삭제 기능 추가
- Implemented image insertion at cursor position / 커서 위치에 이미지 삽입 기능 구현
- Increased default memo size and font size (200x200 → 250x250, 14pt → 16pt) / 메모지 기본 크기 및 폰트 크기 증가 (200x200 → 250x250, 14pt → 16pt)
- Added vertical scrollbar to text area / 텍스트 영역에 수직 스크롤바 추가
- Changed title display to first line text / 제목 표시를 첫 번째 줄 텍스트로 변경
- Set default UI language to English / 기본 UI 언어를 영문으로 설정
- Fixed duplicate display issues with text and images / 텍스트와 이미지 중복 표시 문제 해결

## Version 1.2 (Previous) / 버전 1.2 (이전)
### Main Features / 주요 기능
- Post-it style memo application / 포스트잇 스타일 메모 애플리케이션
- Image paste functionality / 이미지 붙여넣기 기능
- Multi-user support / 다중 사용자 지원
- Encryption feature / 암호화 기능
- System tray support / 시스템 트레이 지원
- Multi-language support (Korean/English) / 다국어 지원 (한국어/영어)
